📊 Smartphone Usage & Productivity Dashboard

Project Overview:
This Power BI dashboard analyzes 50,000 user records to study the relationship between smartphone usage, productivity, stress levels, and sleep patterns.

 Objectives
- Understand screen time behavior
- Analyze productivity variations
- Study stress impact by occupation
- Examine sleep trends across age groups

Key Metrics:
- Average Daily Phone Hours
- Average Productivity Score
- Average Stress Level
- Average Sleep Hours
- Total Users

Tools Used:
- Power BI
- DAX
- Data Modeling
- CSV Dataset

Insights:
- Users average 6.5 daily phone hours
- Productivity & stress levels show minimal variation across occupations
- Sleep duration remains consistent across age groups
- Distribution-level analysis reveals deeper behavioral segmentation
